# Shelley Guitars

A boutique guitar building brand with an interactive site featuring Po the mascot.

## Project Structure

- `web/`: Next.js 14 application.
- `godot/`: Godot 4.3+ source projects for mini-games and site navigation.

## Development

### Web

```bash
cd web
npm install
npm run dev
```

### Godot

Godot projects are located in the `godot/` directory. Export them to `web/public/games/` to see changes in the web app.
